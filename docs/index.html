Quick Install

Auto-detects your platform and installs the appropriate version:

curl -fsSL get.nimsforest.com | sh

For Linux systems (Ubuntu, Debian, CentOS, Alpine, etc.):

curl -fsSL get.nimsforest.com/linux | sh

For macOS (Intel and Apple Silicon):

curl -fsSL get.nimsforest.com/macos | sh

For Windows (PowerShell):

irm get.nimsforest.com/windows | iex
Note: Run this in PowerShell (not Command Prompt). You may need to restart your terminal after installation.

For Go developers (requires Go 1.24+):

go install github.com/nimsforest/nimsforestpackagemanager/cmd@latest
